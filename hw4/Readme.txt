Вивести товари на сторінку у вигляді карток. Оформлення — довільне.

Для доступу до елементів використовуйте
document.querySelector()

Для підготовки/трансформації масиву використовуйте метод
map()

Рекомендовану структуру картки показано на скріншоті.

Для виводу картки використовуйте
innerHTML
